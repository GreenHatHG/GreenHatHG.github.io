样例2

为什么输出没有Myparent2 static block呢？

常量在编译阶段会存入到调用这个常量的方法所在的类的常量池中，本质上，调用类并没有直接引用到定义常量的类，因此并不会触发定义常量的类的初始化

注意：这里指的是将常量放到了Main的常量池中，之后Main与MyParent2就没有任何关系了.

甚至可以将MyParent2的class文件删除
