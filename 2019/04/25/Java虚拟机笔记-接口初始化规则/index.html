接口初始化规则

当java虚拟机初始化一个类时，要求它的所有父类都已经初始化，但是这条规则不适用于接口

  • 在初始化一个类时，并不会先初始化它所实现的接口
  • 在初始化一个接口时，并不会先初始化它的父接口

因此，一个父接口并不会因为它的子接口或者实现类的初始化而初始化，只有当程序首次使用特定接口的静态变量时，才会导致该接口的初始化
